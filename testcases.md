## TC-01 - Фильтр "Диапазон цен"
Предусловие: открыта главная страница, в списке есть объявления с разными ценами.

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Ввести значение в поле "Цена от" | Поле принимает числовое значение |
| 2 | Ввести значение в поле "Цена до" | Поле принимает числовое значение |
| 3 | Применить фильтр (нажать Enter или кнопку применения) | Список обновляется |
| 4 | Проверить каждое объявление в выдаче | Цена каждого объявления >= "от" и <= "до" |
| 5 | Очистить оба поля | Список возвращается к исходному состоянию |

Проверки:
- Все отображаемые цены попадают в заданный диапазон.
- При неправильном диапазоне (от > до) показывается сообщение об ошибке или фильтр не применяется.

## TC-02 - Сортировка "По цене"
Предусловие: открыта главная страница, в списке >= 2 объявлений.

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Выбрать сортировку "По цене (возрастание)" | Список перестраивается |
| 2 | Считать цены всех карточек | Цены идут в порядке возрастания |
| 3 | Выбрать сортировку "По цене (убывание)" | Список перестраивается |
| 4 | Считать цены всех карточек | Цены идут в порядке убывания |

Проверки:
- Карточки расположены в нужном порядке(возрастание/убывание) 

## TC-03 - Фильтр "Категория"
Предусловие: открыта главная страница, существует >= 1 категории с объявлениями.

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Выбрать произвольную категорию из фильтра | Список обновляется |
| 2 | Проверить каждую карточку в выдаче | Категория каждого объявления совпадает с выбранной |
| 3 | Сбросить фильтр категории | Отображаются объявления всех категорий |
Проверки:
- Нет карточек с другой категорией.
- После сброса количество карточек >= количества при активном фильтре.

## TC-04 - Тогл "Только срочные"
Предусловие: открыта главная страница.

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Запомнить количество объявлений без фильтра | - |
| 2 | Включить тогл "Только срочные" | Список обновляется |
| 3 | Проверить каждую карточку | На каждой карточке присутствует метка "Срочно" |
| 4 | Выключить тогл | Список возвращается к состоянию без фильтра |

Проверки:
- Все карточки в активном состоянии тогла имеют признак срочности.
- После выключения тогла список не содержит принудительного ограничения по срочности.

## TC-05 - Контейнер управления таймером на странице Статистики
Предусловие: открыта страница `/stats`.

### TC-05a - Кнопка "Обновить"

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Зафиксировать текущее состояние данных на странице | - |
| 2 | Нажать кнопку "Обновить" | Запрос к серверу отправляется, данные обновляются |
| 3 | Дождаться завершения запроса | Страница отображает актуальные данные без полной перезагрузки |

### TC-05b - Кнопка остановки таймера

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Убедиться, что таймер активен (кнопка "стоп" доступна) | - |
| 2 | Нажать кнопку остановки | Таймер останавливается, кнопка меняет состояние |
| 3 | Подождать интервал автообновления | Данные не обновляются автоматически |

### TC-05c - Кнопка запуска таймера

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Остановить таймер (TC-05b) | - |
| 2 | Нажать кнопку запуска | Таймер запускается, кнопка меняет состояние |
| 3 | Подождать интервал автообновления | Данные обновляются автоматически |

Проверки:
- После нажатия "Обновить" timestamp/данные на странице меняются.
- В остановленном состоянии автообновление не происходит за интервал + буфер.
- В запущенном состоянии автообновление происходит в пределах интервала + буфер.

## TC-06 - Переключение тёмной/светлой темы 

Предусловие: браузер имитирует мобильное устройство.

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть главную страницу | Отображается в одной из тем |
| 2 | Найти переключатель темы | Элемент видим и доступен |
| 3 | Переключить тему (light → dark) | CSS-класс или атрибут `data-theme` меняется на dark |
| 4 | Визуально / через CSS-переменные проверить фон | Фон становится тёмным |
| 5 | Переключить тему обратно (dark → light) | Фон становится светлым |

Проверки:
- Атрибут темы (`data-theme`, класс `dark` и т.п.) принимает ожидаемое значение после каждого переключения.
- Переключатель остаётся интерактивным в обоих состояниях.
